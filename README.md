# Trip Planner

### 注意事項

開發時請使用 develop 分支或從 develop 再開新的個人分支

```javascript
//建立分支
git branch <新增分支名>
//切換分支
git checkout <切換分支名>
```

push 時也請指定開發用分支 develop 或自己的個人分支

```javascript
git push origin develop
```

commit 時請在最後面加上個人可識別代號

```javascript
git commit -m "~訊息~ 可識別代號"
```
